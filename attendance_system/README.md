# Face Recognition Attendance System

A complete web-based attendance system using facial recognition technology.

## Features

- **Face Registration**: Register new faces with multiple image captures
- **Real-time Attendance**: Mark attendance using webcam face detection
- **Database Storage**: Secure SQLite database for attendance records
- **Export Functionality**: Export attendance data to CSV
- **Responsive Design**: Mobile-friendly web interface
